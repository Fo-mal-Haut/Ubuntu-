# 安装方式

前端是**图形化工具安装**，后端则是多种安装方式的聚合，实际情况由软件来源确定

| 火星应用商店中的软件来源         | 它实际调用的底层安装命令           | 本质上的安装方式      |
| -------------------- | ---------------------- | ------------- |
| **Ubuntu官方仓库/PPA仓库** | `sudo apt install 包名`  | **APT安装**     |
| **Snap 包**           | `sudo snap install 包名` | **Snap安装**    |
| **Flatpak 包**        | `flatpak install 包名`   | **Flatpak安装** |
| **下载的 .deb 包**       | `sudo dpkg -i 包名.deb`  | **dpkg安装**    |
