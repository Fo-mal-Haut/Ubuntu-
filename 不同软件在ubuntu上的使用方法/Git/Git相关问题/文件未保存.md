# 本地配置/缓存文件
```text
git merge ubuntu 
更新 a07fd98..5ca59a5 error: 您对下列文件的本地修改将被合并操作覆盖： .obsidian/plugins/daily-statistics/data.json .obsidian/workspace.json 请在合并前提交或贮藏您的修改。 
正在终止
```

**解释**：工作目录（working directory） 有未提交的修改，而合并操作需要覆盖这些文件，Git 为了保护你的工作，拒绝执行合并。

## 文件解释

### **1. `.obsidian/workspace.json`**

- **作用**：保存**当前工作区状态**
    
- **包含内容**：
    - 当前打开的笔记标签页
    - 编辑器分割布局
    - 面板位置和大小
    - 当前光标位置
    - 搜索查询历史


### **2. `.obsidian/plugins/daily-statistics/data.json`**

- **作用**：**Daily Statistics 插件的数据文件**
    
- **包含内容**：
    - 每日编辑统计
    - 字数变化记录
    - 使用习惯数据

## 解决措施：添加到`.gitignore`文件

### **步骤1：创建/编辑 .gitignore 文件**

```bash
# 创建 .gitignore（如果不存在）
touch .gitignore

# 编辑 .gitignore
nano .gitignore
```

### **步骤2：添加 Obsidian 相关规则**

在 `.gitignore` 中添加：
```gitignore
# Obsidian 配置文件
.obsidian/workspace.json
.obsidian/plugins/*/data.json
.obsidian/plugins/daily-statistics/
.obsidian/core-plugins.json
.obsidian/hotkeys.json
.obsidian/appearance.json
.obsidian/app.json
.obsidian/graph.json
```
### **步骤3：删除已跟踪的文件**
```bash
# 从 Git 中删除但保留本地文件
git rm --cached .obsidian/workspace.json
git rm --cached .obsidian/plugins/daily-statistics/data.json
```
### **步骤4：提交 .gitignore 更改**

```bash
git add .gitignore
git commit -m "chore: 添加 Obsidian 个人配置文件到 .gitignore"
```
