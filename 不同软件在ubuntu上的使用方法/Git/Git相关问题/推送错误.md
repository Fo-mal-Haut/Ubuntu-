# 无法访问403
```text
git push origin master remote: Write access to repository not granted. 
fatal: 无法访问 'https://github.com/XXXX/XXXX.git/'：The requested URL returned error: 403
```
### 1. **没有写入权限（最常见）**
- 正在推送一个不属于当前账号的仓库
- 或者是fork的仓库但未设置权限
[[Github密钥权限]]
### 2. **认证问题**
- GitHub token/密码过期
- 使用了错误的凭据

### 3. **仓库访问限制**
- 仓库是私有的且未授权
- 组织仓库有访问限制
