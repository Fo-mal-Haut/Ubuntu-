##### 1.公式输入
行内公式：位于两个$之间
行间公式：位于两个$ $之间

##### 2.求和符号$\sum$
行内公式：\sum_{下标}^{上标}
行内公式上下限位于求和符号上下方：\displaystyle\sum_{下标}^{上标}

##### 3.箭头$\Uparrow$
箭头上标注：X\stackrel{F}{\longrightarrow}Y

##### 4.代码输入
要将单词或短语表示为代码，请将其包裹在反引号 (`` ` ``) 中。
`pip list`

如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号(` `` `)中。
``pip list`内部可以用反引号包围` ``

多行代码使用三个反引号(`` ``` ``)
```
pip list
pip install numpy
```
