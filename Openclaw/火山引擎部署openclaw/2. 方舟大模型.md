### 步骤二：开通方舟大模型服务并获取配置

您可以参考本节指引，开通方舟大模型服务并获取API Key。

1. 请登录[开通管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&advancedActiveKey=model)，选择您期望为OpenClaw提供推理服务的模型家族。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_e332886dd19defda2d113b56be091778.png)

2. 在左侧目录树选择“系统管理 > API Key 管理”，创建并获取API Key。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_3292297fa1de5b34f527c637ef947ce5.png)

3. 在左侧目录树选择“智能广场 > 模型广场”，单击您期望OpenClaw使用的具体模型进入其详情页。本文以“Doubao-Seed-Code”为例。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_b657205398a3b8511a2d6632f218c050.png)

4. 单击复制按钮，获取模型的“Model ID”。
