### 步骤四：配置OpenClaw飞书AI助手

## 1. 云服务器配置

1. [登录云服务器实例](https://www.volcengine.com/docs/6396/67747)。

2. 执行如下命令，配置模型和飞书集成的参数。

```Bash
curl -fsSL https://openclaw.tos-cn-beijing.volces.com/setup.sh | bash -s -- \

--ark-coding-plan "true" \

--ark-model-id "doubao-seed-code" \

--ark-api-key "请替换为方舟API Key" \

--feishu-app-id "请替换为飞书APP ID" \

--feishu-app-secret "请替换为飞书APP Secret"
```

>说明

请根据实际情况，调整命令中如下参数取值：

- --ark-coding-plan：用于指定是否使用您订阅的Coding Plan。

- 若您已经订阅了Coding Plan，请传入true。

- 若您未订阅方舟Coding Plan，请传入false。建议您[订阅方舟Coding Plan](https://www.volcengine.com/activity/codingplan)，包月低至9.9元，模型调用自由。

- --ark-model-id：用于指定您期望OpenClaw使用的方舟模型。

- 若--ark-coding-plan参数配置为true，请将doubao-seed-code替换为Coding Plan支持的模型名称。支持：doubao-seed-code、glm-4.7、deepseek-v3.2、kimi-k2-thinking、kimi-k2.5。

- 若--ark-coding-plan参数配置为false，请将doubao-seed-code替换为您期望使用的方舟模型ID。

- --ark-api-key：用于指定方舟模型API Key。

- --feishu-app-id：用于指定飞书机器人App ID。

- --feishu-app-secret：用于指定飞书机器人App Secret。

## 2.飞书开发配置

1. 登录[飞书开发者平台](https://open.larkoffice.com/app?lang=zh-CN)。 

2. 选择“企业自建应用”页签，并单击您创建的飞书开放应用。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_ee895f1f3bd3fa3c0f201895612fe51b.png)

3. 在左侧目录树选择“开发配置 > 事件与回调”。

- 事件配置

	1. 选择“事件配置”页签，单击“订阅方式”旁的编辑按钮。

	2. 选择“使用 **长连接** 接收事件”，并单击“保存”按钮。

	3. 在“已添加事件”区域，单击“添加事件”按钮。

	4. 在添加事件对话框中，选择“应用身份订阅”页签，并勾选“接收消息”及其它您需要订阅的事件，单击“确认添加”按钮。

- **（可选）** 若您期望将机器人添加到聊天群组中使用，需开通获取群组消息权限。否则，请跳过本步骤。

	1. 在“已添加事件”模块中，单击“接收消息”事件“所需权限”列的“查看其它权限”按钮。

	2. 单击“获取群组中所有消息（敏感权限）”。

	3. 在弹窗中单击“开通权限”按钮，申请开通。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_caa3d7063af9b35666933809a03e6a1a.png)

- 回调配置

	1. 选择“回调配置”页签，单击“订阅方式”旁的编辑按钮。

	2. 选择“使用 **长连接** 接收回调”，并单击“保存”按钮。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_130cf2011c8b9dcd30e81e41f9b602ee.png)

## 3. 版本发布

1. 新建机器人版本并发布。

2. 单击顶部的“创建版本”按钮。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_1058ca45da12bc3af800a782257e76e5.png)

3. 按需配置应用版本号、默认能力及更新说明等信息。[了解更多](https://open.larkoffice.com/document/best-practices/intro-to-custom-app-review)。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_0d5aed2b0fec9906af56a0284f656c75.png)

4. 单击页面底部的“保存”按钮，创建版本。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_f5e63e15c8b10e13cc1be20b93841fd1.png)

5. 单击页面右上角的“确认发布”按钮，完成应用发布。

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_25550af4ea9aea79198c7c75d756f36f.png)

6. 待管理员通过发布审核后，即可正式在飞书中使用您的定制化私人AI助手。
