在Linux中，终端分页查看器是命令行工具，让你可以**逐页查看文本文件**，而不会让内容快速滚过屏幕。这里介绍终端分页查看器中两种最常用的方法：`less`命令和`more`命令。
# 1. 使用`less`命令

```Bash
# 基本使用
less 文件名

# 常用操作
空格键   # 向下翻一页
b        # 向上翻一页
/关键词  # 搜索内容
q        # 退出
```

# 2. 使用`more`命令

`more`命令也是用来分页显示文件内容的，但它的功能相比`less`要少一些。使用`more`的基本步骤如下：

```Bash
# 基本使用
more 文件名

# 常用操作
空格键   # 向下翻一页
Enter   # 逐行滚动
q        # 退出
```
