**macOS 终端命令与 Linux 高度相似**，因为它们都源于 **UNIX** 系统，因此对于Linux系统的大多数终端命令，都可以直接沿用到MacOS上。因此所有终端命令，都可参阅[Linux终端命令导航](../../Ubuntu/Linux基本知识/终端/Linux终端命令导航.md)

---

# **macOS 终端 vs Linux 的主要相似点**

| 方面 | macOS (zsh) | Linux (bash) | 相似度 |
|------|-------------|--------------|--------|
| **基本命令** | `ls`, `cd`, `cp`, `mv`, `rm`, `cat`, `grep` 等 | 完全一致 | ⭐⭐⭐⭐⭐ |
| **文件系统结构** | 根目录 `/`，用户目录 `/Users/用户名` | 根目录 `/`，用户目录 `/home/用户名` | ⭐⭐⭐⭐☆ |
| **包管理** | Homebrew (`brew`) | apt/yum/pacman (`apt-get`) | ⭐⭐⭐☆☆（概念相似，命令不同） |
| **权限系统** | `chmod`, `chown`, `sudo` | 完全一致 | ⭐⭐⭐⭐⭐ |
| **环境变量** | `~/.zshrc` 或 `~/.bash_profile` | `~/.bashrc` | ⭐⭐⭐⭐☆（配置文件不同） |

## **`ls` 命令的常用选项（与 Linux 相同）**

| 选项 | 作用 | 示例 |
|------|------|------|
| `ls` | 基本列表（不显示隐藏文件） | `ls` |
| `ls -l` | 详细列表（权限、大小、时间） | `ls -l` |
| `ls -a` | **显示所有文件**（包括以`.`开头的隐藏文件） | `ls -a` |
| `ls -la` | **详细列表+显示所有文件**（最常用） | `ls -la` |
| `ls -lh` | 人类可读的文件大小（KB, MB, GB） | `ls -lh` |
| `ls -t` | 按修改时间排序（最新在前） | `ls -lt` |
| `ls -R` | 递归列出子目录内容 | `ls -R` |

macOS 终端有强大的**自动补全**功能：
```bash
# 输入部分路径后按 Tab
cd /Us⇥  # 自动补全为 /Users/
cd ~/Do⇥ # 自动补全为 ~/Documents/
```
