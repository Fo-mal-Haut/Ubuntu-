Anthropic 提供了丰富的开源 Skills 示例，涵盖多个领域：

**文档处理**：

- **doc-coauthoring**: 结构化的文档协作创作工作流
- **docx**: Microsoft Word 文档处理（源码可用）
- **pdf**: PDF 表单填写和处理（源码可用）
- [[官方Skills/pptx]]: PowerPoint 演示文稿创建（源码可用）
- **xlsx**: Excel 电子表格处理（源码可用）

**创意与设计**：

- **frontend-design**: 创建独特的生产级前端界面
- **canvas-design**: Canvas 图形设计（包含54种字体）
- **algorithmic-art**: 生成式算法艺术
- **theme-factory**: 10种预设设计主题
- **slack-gif-creator**: Slack GIF 动画创建

**技术开发**：

- **mcp-builder**: Model Context Protocol 服务器构建
- **skill-creator**: Agent Skills 创建工具
- **web-artifacts-builder**: Web Artifacts 构建
- **webapp-testing**: Web 应用自动化测试

**企业与通信**：

- **internal-comms**: 企业内部沟通模板
- **brand-guidelines**: 品牌一致性指导

完整源码和详细文档，请访问 [Anthropic Skills 仓库](https://github.com/anthropics/skills)。
