# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 仓库类型

这是一个 **Obsidian 知识库**，包含关于 Ubuntu Linux、Git、软件开发工具及相关技术的 Markdown 文档。这不是一个软件项目——没有构建命令、测试或源代码。

## 文档结构

- `Ubuntu相关软件/` - 工具文档（Claude Code、Git、Node.js、Ruby、Obsidian）
- `软件的安装与卸载/` - Ubuntu 软件安装指南
- `Ubuntu/` - Ubuntu 系统文档和 Linux 命令
- `Github相关/` - GitHub 操作指南
- `Markdown相关/` - Markdown 参考资料
