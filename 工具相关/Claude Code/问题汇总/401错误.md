# 问题解释与解决

`401 {"error":{"message":"token is unusable"}` 错误表示：

1. **API密钥无效或已过期**
2. **API密钥格式不正确**
3. **权限不足**或密钥被禁用

原因是在cc- switch中选择了错误的API，而这个API已经被我删除了，需要重新生成一个新的API
