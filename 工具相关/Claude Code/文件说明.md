# Claude相关文件

主要有三类
```
.claude
.claude.json
.claude.json.backup.*
```

## 一、`.claude/` —— Claude 的“主工作目录”

**`.claude`**

这是 **Claude 在用户目录下的核心数据目录**，地位类似于：

- `.vscode/`
- `.npm/`
- `.ssh/`

里面通常会包含（不同版本略有差异）：

- 本地会话缓存
- 账号/设备标识（token 不一定是明文）
- Claude CLI / 桌面应用的运行状态
- 本地索引或临时文件

## 二、`.claude.json` —— Claude 的核心配置文件

 **`.claude.json`**是一个 **JSON 格式的配置文件**，作用类似：

- `.npmrc`
- `.gitconfig`
- `.vscode/settings.json`

通常包含：

- Claude 使用的模型偏好
- 本地功能开关
- CLI / App 的运行参数
- 实验性功能 flag
- 账号关联的某些本地状态

## 三、`.claude.json.backup.` —— 自动备份

 **`.claude.json.backup.1770602217532`**

- Claude **每次修改或升级配置**
- 或每次检测到 `.claude.json` 结构变化
- 或版本迁移
就会自动生成一个 **时间戳备份**
