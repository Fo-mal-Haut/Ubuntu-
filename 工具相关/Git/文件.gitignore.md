# 忽视所有子文件夹下的不可见文件

**1.所有`DS_Store`文件

```.gitignore
# macOS 系统文件
.DS_Store
**/.DS_Store
```

**2.所有`.claude`文件夹**
```.gitignore
**/.claude/
```

**注意**：末尾的 `/` 确保只匹配目录，不匹配文件

## 不同写法的区别

| **写法**                                      | **含义**               | **例子**                       |
| ------------------------------------------- | -------------------- | ---------------------------- |
| `.obsidian/plugins/*`                       | 忽略 plugins 下的一级内容    | 匹配：`plugins/a/`，`plugins/b/` |
| `.obsidian/plugins/**`                      | 递归忽略所有内容             | 匹配：`plugins/a/deep/file`     |
| `.obsidian/plugins/*.json`                  | 忽略 plugins 下的一级 JSON | 匹配：`plugins/config.json`     |
| `!.obsidian/plugins/community-plugins.json` | 不忽略这个文件（例外）          | 保留这个特定文件                     |
