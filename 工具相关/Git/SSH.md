# 公钥和私钥

## **1. 私钥（identification）**

- **文件**：`/Users/fomalhaut/.ssh/id_rsa`
- **作用**：**你的数字身份证**，必须**绝对保密**（类似银行卡密码）
- **用途**：
    - 本地认证：向服务器证明“你就是你”
    - 解密：解密用对应公钥加密的信息
- **重要性**：**绝对不能分享给任何人**，丢失或泄露需要立即撤销
## **2. 公钥（public key）**

- **文件**：`/Users/fomalhaut/.ssh/id_rsa.pub`
- **作用**：**你的公开身份信息**（类似公开的银行账号）
- **用途**：
    - 添加到 GitHub、GitLab、服务器等
    - 加密：其他人可以用它加密信息，只有你能解密
    - 验证：服务器用它验证你的私钥
- **特性**：可以安全地分享给任何人
